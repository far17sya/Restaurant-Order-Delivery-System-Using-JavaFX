**Restaurant Order Delivery System Using JavaFX**
This project is a Restaurant Order & Delivery System developed using JavaFX. The system allows users to browse a food menu, add items to a cart, proceed with checkout, receive a receipt, and rate their experience. It demonstrates core OOP principles, GUI design, and exception handling in Java.

**Project Overview**
The system was designed to simplify food ordering in a restaurant environment by replacing manual order-taking with an interactive digital interface. Customers can:
- View available food items
- Select items and adjust quantities
- Add items to a shopping cart
- Proceed through a checkout process
- View a generated receipt
- Submit feedback through a rating system

**Features**
1. User Login Interface
Basic login form using JavaFX layouts
Validates user inputs

2. Menu & Cart System
Displays list of food items
Users can view item details
Add items with desired quantity
Automatically calculates total price

3. Checkout & Payment Page
User enters name, phone number, and address
Selects payment method (Online Banking, TnG eWallet, Credit/Debit)

4. Receipt Generation
Shows ordered items with quantity and cost
Displays total amount
Presented in a separate window
